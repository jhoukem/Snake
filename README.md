# Snake
A simple old snake game in C developped for learning purpose.  

## Installation guide
Simply compiles the sources:  
```
$ git clone http://github.com/jhoukem/Snake
$ cd Snake
$ make
```
Then launch the game:   
`$ ./snake 10`
